1.1 Definitions

1.1.1 Acquisition Value = Standalone Value + Merger Synergies   Definition

1.1.2 Fair Market Value = based on operation company + Merger synergies in group of buyers   Definition

1.2 Value Created by an Acquisition

NPVacq = Wacq - P or the value created by and acquisition is equal to the PV of cashflows of acquired firm less the price paid for it.

1.2.1 Differential approach

  1. Effect on cash flow
    • Incremental flows due to collusive policies
    • Incremental flows due to gains in operational efficiency
    • Incremental flows due to gains of intangible natures(marketing etc)
  2. Effect on risk profile
    • Positive effects: Leverage over suppliers/buyers
    • Negative effects: in case of vertical integration
  3. Effect on credit profile
    • If debts increases, so does EV due to tax shields.

1.2.2 Benifits

Wacq = Wbase + Ws + Wo where

  • Wbase = base value
  • Ws = value of synergies
  • Wo = value of opportunities created by the acquisition
